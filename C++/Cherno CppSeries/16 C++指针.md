# C++指针

> 对内存的控制是至关重要的
>
> 所有指针都是**保存内存地址的整数**



```cpp
#include <iostream>
#define LOG(x) std::cout<<x<<std::endl

int main(){
    int var = 8 ;
    void* ptr1 = &var;
    
    
    //类型对指针的值没有作用，但是类型对内存操作有用
    //类型影响指针的字节长度
    double* ptr2 = (double*)&var;
    
    *ptr1 = 10;//会报错，因为ptr1的类型是void，计算机不知道对应的字节长度
```
